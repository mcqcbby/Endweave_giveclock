Bell Giving Plugin

Introduction
The Bell Giving Plugin automatically checks if a player has a bell in their inventory when they join the server. If the player does not have a bell, it will give the player a bell, and the bell cannot be dropped.

Installation
After downloading the plugin, place it in the plugins folder and then restart the server.

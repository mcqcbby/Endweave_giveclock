Give Clock Plugin

Introduction
The Give Clock plugin automatically checks if a player has a clock in their inventory when they join the server. If they do not have one, the plugin gives them a clock, which cannot be dropped.

Installation
Download the plugin and place it in the plugins folder, then restart the server.
